Es proyecto consiste en una API para un sistema de reservas, las funciones principales con las crear, editar, borrar y ver todas las reservas realizadas.

Al momento de crear una reserva se tiene que tener en cuenta que no debe coincider con alguna fecha o hora de otra reserva, si no la APP arrjará un error y no dejará agendarla. Esto también aplica para la edición de reservas.

Se recomienda probar las funciones de manera local antes de desplegarlas.

A continuación dejaré una web la cual tiene implementada dichas funciones.

https://proyectoappreservas.000webhostapp.com/

Al momento de ejecutar el programa de manera local por medio de la consola resivirá los enlaces para consumir cada función.
